# 简介

在本简介中，我们将涵盖两个关键主题，为你使用 Rust 开发 Espressif 设备打下坚实基础：

1. **[硬件概述](./hardware-overview.md)**：了解 Espressif 生态系统，包括不同的 SoC 系列、架构和开发板。这些知识将帮助你为项目选择合适的芯片，并了解你将要编程的底层硬件。

2. **[辅助包概述](./ancillary-crates.md)**：探索包（Crate）的生态系统，这些包提供了从内存分配到无线通信等所有功能的构建模块。

了解硬件能力和软件生态系统将为你准备好[快速入门](../getting-started/index.md)章节中的实践内容，在那里你将设置开发环境并创建第一个应用程序。

让我们从你将要使用的硬件概述开始。
